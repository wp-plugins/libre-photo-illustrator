== Description ==

Illustrator for free/libre photos that are in the public domain or at least published under one of the CC-BY licenses. This sidebar widget has the aim to enrich your weblog with selectable and configurable OpenContent materials like drawings, animations and other kinds of images.

== Installation ==

1. Download the plugin
2. Unzip the package to the directory "wordpress/wp-content/plugins/"
3. Activate plugin in [Admin Menu] -> [Plugins]
4. Activate widget in [Admin Menu] -> [Design] > [Widgets]
5. Configure options in [Admin Menu] -> [Settings] -> [Libre-Photo-Illustrator]

== Frequently Asked Questions ==

= How to change look and feel? =

Simply change settings in stylesheet style.css.

= How to change look and feel in the backend? =

Simply change settings in stylesheet style2.css.

= What languages are supported? =

english, german and spanish
